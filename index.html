Uframer Android UI Library by uframer

Mimic of Windows Phone 7 UI controls on Android. It's in its very early stage.

Install

Use it like any other ADT Library Projects in Eclipse.

Download

You can download this project in either zip or tar formats.

You can also clone the project with Git by running: 

$ git clone git://github.com/uframer/Uframer-Android-UI-Library

get the source code on GitHub : uframer/Uframer-Android-UI-Library
